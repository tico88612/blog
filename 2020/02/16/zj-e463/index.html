題目 Problem

題目連結：https://zerojudge.tw/ShowProblem?problemid=e463

敘述 Description

給你一個數字n(n<=100)，可以在n旁邊加上<=n/2的數字，一直加到不能加為止

比如說n=6，可以變成16,26,36

26可以變成126

36可以變成136

最後總共有6,16,26,36,126,136共六個數字

輸入 Input

多筆輸入

每行為n(n<=100)

輸出 Output

輸出共有幾個數字

再輸出每個數字，記得要排序

(有一些數字雖然會重複，但因為是由不同數字變出來的，所以都要輸出)
