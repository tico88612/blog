題目 Problem

敘述 Description

現在地板上有兩個立方體和一面垂直於地板的牆壁

離牆壁較遠的立方體以等速撞上另一個立方體,使其撞上牆壁並以原路徑反彈

假設沒有摩擦力且所有碰撞均為完美彈性碰撞(也就是沒有能量散失)

且離牆壁較遠的立方體的質量是離牆壁較近的立方體的10x倍

給你x,請求出總共會碰撞幾次

輸入 Input

每一行一個非負偶數x(x<923)

輸出 Output

總共會碰撞幾次
